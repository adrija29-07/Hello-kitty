## 🎀 Hello Kitty Dress-Up Game

A cozy, pixel-style dress-up web game where you can style your character and explore different aesthetic backgrounds 🌸✨

Switch outfits, change accessories, and enjoy relaxing vibes across multiple scenic environments 🌙🏖️🌼

## Key Features
Character customization with multiple outfit options
Accessory selection and visual updates in real-time
Multiple background themes (night, beach, morning)
Structured and responsive UI layout
Lightweight and fast-loading frontend

## 🌅 Background Themes

Your game includes beautiful pixel environments:

🌙 Night Mode – calm village with glowing lights
🏖️ Beach Mode – sunny tropical vibe
🌼 Morning Meadow – soft pastel mountains & flowers
